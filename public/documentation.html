Movie API Project Documentation Page

This project will present a movie API. It will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

App Features

  • Return a list of all movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info(username)
  • Allow users to add a movie to their list of favorites, showing only a text that a movie has been added
  • Allow users to remove a movie from their list of favorites, showing only a text that a movie has been removed
  • Allow existing users to deregister, showing a text that a user email has been removed

Api Endpoints

Allow new users to register /users Post - A JSON object holding data about the users to add with the following structure: { "name" : "testname", "Password": "testpassword", "Email": "testemail@testdomain.com", "Birthday": "01.01.1995" } A JSON object holding data about the user that was added with the following structure: { "name" : "testname", "Password": "testpassword", "Email": "testemail@testdomain.com", "Birthday": "01.01.1995" }
Allow users to update their user info(username) /users/:id Put :id - A JSON object holding data about the updated information
Allow users to add a movie to their list of favorites, showing only a text that a movie has been added /users/:id/favorites/:movieName Post :id, :movieName - A JSON object holding data about the user and the movie that was added
Allow users to remove a movie from their list of favorites, showing only a text that a movie has been removed /users/:id/favorites/:movieName Put :id, :movieName - A JSON object holding data about the user and the movie that was removed
Allow existing users to deregister, showing a text that a user email has been removed /users/:id Delete :id - A message indicating that the user account has been deleted
